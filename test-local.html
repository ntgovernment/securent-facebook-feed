üé® Configuration Options

Available Data Attributes:

<div 
  data-securent-fb-widget
  data-api-url="mock-data.json"
  data-items-per-page="5"
  data-fallback-url="https://www.facebook.com/SecureNT"
  data-theme="light"
></div>
Attribute Default Description
data-api-url SecureNT API API endpoint URL
data-items-per-page 5 Posts per page
data-fallback-url Facebook page Alternative link when API fails
data-theme light Theme: light or dark

üîç Browser Console Commands

Get Widget Instance:

const widget = SecureNTFacebookWidget.getInstance(
  document.querySelector('[data-securent-fb-widget]')
);
console.log(widget);

Check Version:

console.log(SecureNTFacebookWidget.version);

Clear Storage:

// Clear cache
localStorage.removeItem('securent-fb-cache');
localStorage.removeItem('securent-fb-cache-time');

// Reload page
location.reload();

Reinitialize:

SecureNTFacebookWidget.init();
