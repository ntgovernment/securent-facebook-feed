SecureNT Facebook Feed Widget - Example

Default Configuration

This widget uses all default settings (5 items per page, light theme).

Custom Configuration

This widget displays 3 items per page with custom fallback URL.

Integration Instructions

For Squiz Matrix:

Add the following code to your page within the <section class="facebook-feed"> section:

<!-- Add to page HEAD or before closing BODY tag -->
<link rel="stylesheet" href="https://your-filebridge-url/dist/securent-fb-widget.css">

<!-- Add inside <section class="facebook-feed"> -->
<div class="container">
  <div data-securent-fb-widget></div>
</div>

<!-- Add before closing BODY tag -->
<script src="https://your-filebridge-url/dist/securent-fb-widget.min.js"></script>

Configuration Options:

  • data-api-url - API endpoint (default: SecureNT API)
  • data-items-per-page - Number of posts per page (default: 5)
  • data-fallback-url - Fallback URL (default: https://www.facebook.com/SecureNT)
  • data-theme - Theme: light or dark (default: light)

Example with all options:

<div 
  data-securent-fb-widget
  data-items-per-page="10"
  data-fallback-url="https://www.facebook.com/SecureNT"
  data-theme="light"
></div>
